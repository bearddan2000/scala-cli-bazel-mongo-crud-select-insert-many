# scala-cli-bazel-mongo-crud-select-insert-many

## Description
Creates a user with credentials
and lists collection `test`.

## Tech stack
- scala
- bazel
  - mongo connector
- mongo

## Docker stack
- mongodb:latest
- openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
